Base de Conhecimento

Para gerar relatórios personalizados no sistema LivroTech, siga os passos abaixo:

  1. Acesse o menu "Relatórios" na barra lateral.
  2. Selecione o tipo de relatório desejado no campo "Tipo de Relatório".
  3. Defina o período desejado nos campos "Data Início" e "Data Fim".
  4. Escolha como deseja agrupar os dados no campo "Agrupar por".
  5. Clique em "Aplicar Filtros" para gerar o relatório.
  6. Utilize os botões "Exportar" ou "Imprimir" para salvar ou imprimir o relatório gerado.

Você também pode salvar o relatório personalizado para uso futuro clicando no botão "Novo Relatório" na seção "Relatórios Salvos".

Para configurar o backup automático do sistema, siga estas instruções:

  1. Acesse o menu "Configurações" no canto superior direito.
  2. Clique na opção "Backup e Restauração" no menu lateral.
  3. Marque a opção "Habilitar backup automático".
  4. Selecione a frequência desejada (Diário, Semanal ou Mensal).
  5. Defina o horário para execução do backup no campo "Hora do Backup".
  6. Escolha por quanto tempo deseja manter os backups antigos.
  7. Clique em "Salvar Configurações" para aplicar as alterações.

Recomendamos agendar o backup para horários de menor uso do sistema, geralmente durante a madrugada.

O LivroTech possui três tipos principais de usuários com diferentes níveis de permissão:

  • Cliente: Pode visualizar produtos e realizar compras.
  • Funcionário: Pode registrar vendas, gerenciar produtos e atender clientes.
  • Administrador: Tem acesso completo ao sistema, incluindo gerenciamento de usuários, relatórios e configurações.

Para alterar as permissões de um usuário:

  1. Acesse o menu "Usuários" → "Listar Usuários".
  2. Localize o usuário desejado e clique no botão "Editar".
  3. Altere o tipo de usuário no campo "Tipo de Usuário".
  4. Clique em "Salvar" para aplicar as alterações.

Problema: Página não carrega corretamente

Soluções:

  • Limpe o cache do navegador e tente novamente.
  • Tente acessar com outro navegador.
  • Verifique sua conexão com a internet.
Problema: Erro ao gerar relatórios

Soluções:

  • Verifique se o período selecionado é válido.
  • Tente gerar um relatório com menos dados.
  • Reinicie a sessão fazendo logout e login novamente.
Problema: Sistema lento

Soluções:

  • Verifique a quantidade de guias abertas no navegador.
  • Certifique-se de que seu dispositivo atende aos requisitos mínimos.
  • Tente acessar em horários de menor uso.
